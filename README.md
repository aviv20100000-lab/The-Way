# The Way — אפליקציית מעקב תזונה

אפליקציה למאמנים ומתאמנים: המתאמנים מצלמים את הארוחות, המאמן רואה הכל עם ערכים תזונתיים.

## התקנה והרצה

```bash
bun install
bun run dev
```

פתח [http://localhost:3000](http://localhost:3000)

## איך זה עובד

1. **מתאמן** — מתחבר, לוחץ "צלם ארוחה חדשה", מעלה תמונה, בוחר מזון וכמות (בגרמים), ושומר.
2. **מאמן** — רואה את כל הארוחות של המתאמנים שלו, עם תמונות וסיכום קלוריות / חלבון / פחמימות / שומן.
3. המאמן יכול להוסיף מתאמנים חדשים ולסנן לפי תאריך ומתאמן.

## טכנולוגיות

- Next.js 15 + Bun
- SQLite (מסד נתונים מקומי)
- Tailwind CSS
- ממשק בעברית (RTL)
